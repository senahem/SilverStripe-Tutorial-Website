Welcome to SilverStripe


SilverStripe provides a framework built around PHP, with a nice GUI that allows the user to program complex and simple websites. SilverStripe provides easy code production and adaptability because of its Model-view-controller architecture.

Since SilverStripe is an entirely open source CMS platform, it allows collaborative development as well as constant updating of the system by not just the original developer but users also. It is extremely dynamic and has been constantly remodeled and developed to address new issues as well as improve efficiency and user experience.

Technological bases


  • The SilverStripe Framework as with the rest of the platform is written in PHP and takes advantage of object oriented design capabilities. Furthermore, it's based on today's modern programming techniques including model-view-controller and object-relational mapper.

  • SilverStripe generates a markup language using custom templates based on W3C HTML5 which offer simple placeholders and conditional logic.

  • The SilverStripe Framework approach is stated to remove complexity for beginners and also offer more flexibility for experienced developers

    • Requirements:

Strengths and Weaknesses

Advantages:

  • The Framework provides a flexible foundation
  • The template is perfect for complicated and simple layouts
  • The framework is intuitive and beautiful
  • Powerful, easy and consistent interface for managing data
  • Quick and easy installation process
  • Provides Drag-and-drop structuring
  • Low memory requirements (48 MB)
  • The user can control all CMS extensions with the admin panel

Disadvantages:

  • You will need a general understanding of few programming languages and technical aspects unlike other frameworks
  • Premium themes, add-ons, and modules are virtually nonexistent.
  • Very little information available online for the CMS, thus challenging for a beginners
