Conclusion

  • Easy to install and configure.
    • Difficult to troubleshoot errors.
  • Easy to create and edit pages.
  • Performs up to expectations.
  • Not as popular compared to its competitors (ex WordPress).
  • Strikes a good balance between complexity and simplicity.
  • Themes can be added or changed with ease
  • Requires general understanding of a few programming languages for more complex websites (ex HTML,CSS,PHP)
  • Small learning curve compared to HTML & CSS
  • No regrets in choosing SilverStripe.
